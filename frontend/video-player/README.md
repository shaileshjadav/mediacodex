# Video Player

A simple and elegant video player application built with modern web technologies.

## Features

- **Play/Pause Control**: Easy-to-use play and pause functionality
- **Volume Control**: Adjustable volume with mute option
- **Progress Bar**: Visual progress indicator with seek functionality
- **Fullscreen Mode**: Toggle fullscreen viewing experience
- **Responsive Design**: Works seamlessly across different screen sizes
- **Keyboard Shortcuts**: Control playback with keyboard shortcuts
- **Multiple Format Support**: Supports various video formats (MP4, WebM, OGV)
